# SkillSync_Learning_Platform
SkillSync is a multi-user platform that connects learners with mentors and enables peer learning. Users can search for mentors, book mentoring sessions, join learning groups, and review mentors after sessions.

This project will include a robust backend which utilises SpringBoot, Docker, JWT, RabbitMQ and Oracle DB
-will not have a frontend, can be tested using software like Postman
